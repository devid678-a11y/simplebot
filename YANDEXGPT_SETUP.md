# Настройка YandexGPT для парсинга Telegram каналов

## 1. Получение API ключа

1. Перейди на https://yandex.cloud/
2. Создай аккаунт или войди в существующий
3. Создай новый проект
4. Перейди в раздел "API ключи"
5. Создай новый API ключ для YandexGPT
6. Скопируй ключ

## 2. Получение Folder ID

1. В том же проекте перейди в раздел "Ресурсы"
2. Создай новую папку или используй существующую
3. Скопируй ID папки

## 3. Настройка переменных окружения

Создай файл `.env` в папке `functions/`:

```env
YANDEX_API_KEY=твой_api_ключ_здесь
YANDEX_FOLDER_ID=твой_folder_id_здесь
YANDEX_MODEL=yandexgpt
FIREBASE_PROJECT_ID=dvizheon
```

## 4. Активация YandexGPT

1. В консоли Yandex Cloud перейди в раздел "AI"
2. Выбери "YandexGPT"
3. Активируй сервис для твоего проекта
4. Убедись, что у тебя есть доступ к API

## 5. Тестирование

Запусти тест:
```bash
cd functions
node test-parser.js
```

## Важно!

- YandexGPT платный, но есть бесплатный лимит
- Следи за расходом токенов в консоли
- API ключ должен быть действительным
- Folder ID должен существовать в твоем проекте
