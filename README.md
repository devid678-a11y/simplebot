# Простой Telegram Бот

Самый простой Telegram бот, который отвечает на сообщения.

## Запуск локально

1. Установите зависимости:
```bash
npm install
```

2. Установите токен бота:
```bash
# Windows
set BOT_TOKEN=ваш_токен_здесь

# Linux/Mac
export BOT_TOKEN=ваш_токен_здесь
```

3. Запустите бота:
```bash
npm start
```

## Функции

- `/start` - приветствие
- `/help` - помощь
- Отвечает на любые текстовые сообщения

## Деплой

Для деплоя на Timeweb Cloud используйте Dockerfile.
