# Диагностика проблем

## Сервер запущен
✅ Сервер работает на порту 3000
✅ Процесс активен (PID: 1656)

## Что проверить:

### 1. Откройте браузер
Перейдите по адресу: **http://localhost:3000**

### 2. Откройте консоль разработчика
- Нажмите `F12` или `Ctrl+Shift+I`
- Перейдите на вкладку **Console**
- Проверьте, есть ли ошибки (красным цветом)

### 3. Проверьте вкладку Network
- Откройте вкладку **Network**
- Обновите страницу (`F5`)
- Проверьте, все ли файлы загружаются (код 200)

### 4. Возможные ошибки и решения:

#### Ошибка: "Failed to fetch" или "Network error"
**Решение:** Сервер не запущен. Запустите:
```bash
cd architecture-site
npm run dev
```

#### Ошибка: "Cannot find module" или "Module not found"
**Решение:** Переустановите зависимости:
```bash
cd architecture-site
rm -rf node_modules
npm install
npm run dev
```

#### Ошибка: "ERR_CONNECTION_REFUSED"
**Решение:** 
1. Проверьте, что сервер запущен
2. Проверьте порт в `vite.config.js`
3. Попробуйте другой порт (например, 3001)

#### Белый экран
**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте ошибки JavaScript
3. Убедитесь, что все файлы загрузились

#### Страница загружается, но без стилей
**Решение:** Проверьте пути к CSS файлам в консоли браузера

## Быстрое решение

Остановите текущий процесс и перезапустите:

```bash
# Windows PowerShell
Get-Process | Where-Object {$_.Id -eq 1656} | Stop-Process -Force

# Затем запустите заново
cd architecture-site
npm run dev
```

## Альтернативный порт

Если порт 3000 занят, измените в `vite.config.js`:
```js
server: {
  port: 3001,  // измените на другой порт
  host: true,
  open: true
}
```

И откройте: **http://localhost:3001**
