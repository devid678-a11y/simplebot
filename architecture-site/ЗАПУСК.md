# Инструкция по запуску сайта

## Быстрый запуск

1. Откройте терминал в папке `architecture-site`
2. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```

3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

4. Откройте браузер и перейдите по адресу:
   ```
   http://localhost:3000
   ```

## Альтернативный способ (Windows)

Дважды кликните на файл `start.bat` в папке проекта.

## Если порт 3000 занят

Если вы видите ошибку о том, что порт занят:

1. Найдите процесс, использующий порт 3000:
   ```bash
   netstat -ano | findstr :3000
   ```

2. Или измените порт в файле `vite.config.js`:
   ```js
   server: {
     port: 3001,  // измените на другой порт
     host: true,
     open: true
   }
   ```

## Возможные проблемы

### "Cannot find module"
Запустите `npm install` еще раз.

### "Port already in use"
Измените порт в `vite.config.js` или остановите процесс, использующий порт.

### Страница не открывается автоматически
Откройте браузер вручную и перейдите на `http://localhost:3000`

## Структура URL

- Главная: `http://localhost:3000/`
- Блог: `http://localhost:3000/blog`
- Контакты: `http://localhost:3000/contact`
