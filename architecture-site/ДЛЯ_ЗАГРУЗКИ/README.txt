ЭТА ПАПКА СОДЕРЖИТ ВСЕ ФАЙЛЫ ДЛЯ ЗАГРУЗКИ!

НОВОЕ: Добавлена страница просмотра статей!

ИНСТРУКЦИЯ:
-----------

1. Откройте файловый менеджер в панели Timeweb
2. Перейдите в папку public_html
3. Загрузите ВСЕ файлы из этой папки

СТРУКТУРА:
----------

ДЛЯ_ЗАГРУЗКИ/
  ├── index.html          (главная страница)
  ├── _redirects          (редиректы для SPA)
  └── assets/             (стили и скрипты)
      ├── index-C1QNxd8z.css
      └── index-QXw5ak-m.js

✅ Интеграция с WordPress API

✅ Похожие статьи

ВАЖНО:
------
- Загрузите ВСЕ файлы из этой папки
- Сохраните структуру папок (assets должна быть папкой)
- Все файлы должны быть в public_html на сервере

WORDPRESS:
----------
После установки WordPress:
1. Создайте файл .env в architecture-site/
2. Добавьте: VITE_WORDPRESS_API_URL=https://ваш-wordpress/wp-json/wp/v2
3. Пересоберите проект: npm run build
4. Загрузите обновленные файлы

ГОТОВО!
После загрузки сайт будет работать по адресу вашего домена.
